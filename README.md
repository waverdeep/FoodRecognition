# FoodRecognition
사진안에 있는 음식이 어떤 음식인지 인식할수 있는 인공지능 모델 개발

## Getting Started
You can start on any computer that can learn deep learning.

### Prerequisites
```
torch==1.9.0
torchvision==0.9.0
tqdm==4.46.1
tensorboard==2.2.2
tensorflow-gpu==2.2.0
```

### Installing
```
python setup.py
```
